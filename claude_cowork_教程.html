🤖 1   什么是 Claude Cowork？

Claude Cowork 是 Anthropic 推出的一款 桌面 AI 助手，它直接运行在你的电脑上，可以帮你处理本地文件。你不需要学习任何编程——只需用自然语言告诉它你想做什么，它就会帮你完成。

📊

📑

🔍

搜索信息

联网查询最新资讯，帮你快速获取所需知识

📁

读取、整理、重命名、分析你电脑里的文件

💬

随时问答

任何工作问题都可以问它，它会用中文回答

🚀 2   第一步：开始使用

  1. 下载并安装 Claude 桌面应用 访问 claude.ai，下载适合你电脑系统（Windows / macOS）的版本，按提示安装完成。
  2. 注册或登录账号 用邮箱注册一个免费账号，或者用 Google 账号一键登录。
  3. 在应用内找到 Cowork 功能 打开桌面应用后，在侧边栏或主界面找到 Cowork 入口并点击。
  4. 开始对话 界面底部有一个输入框，直接打字描述你的需求即可，不需要任何特殊格式。
✅
无需编程知识你只需要会打字就能使用 Claude Cowork。用中文告诉它你想做什么，它会自动完成。

📂 3   第二步：选择工作文件夹

在处理本地文件之前，你需要告诉 Claude 可以访问电脑上的哪个文件夹。这一步非常重要，也很简单。

  1. 等待 Claude 请求文件夹权限 当你要求 Claude 处理本地文件时，它会主动弹出一个对话框，请求你选择一个文件夹。
  2. 点击「选择文件夹」按钮 在弹出的系统文件选择器中，找到你想让 Claude 访问的文件夹（比如「桌面」或「工作文件」文件夹）。
  3. 确认授权 选择好后点击「确定」。Claude 就可以读取该文件夹里的文件，并将新建文件保存到这里。
💡
建议：新建一个专用文件夹在桌面新建一个名为「AI工作区」的文件夹，专门用来存放让 Claude 处理的文件和它生成的成果，便于管理。

💬 4   第三步：如何与 Claude 对话

使用 Claude Cowork 的核心就是「用自然语言描述你的需求」。以下是一些基本原则：

🎯 说清楚你想要什么结果

你
帮我写一份关于第一季度销售情况的报告
Claude
好的！请问你希望报告包含哪些内容？比如销售数据、主要亮点、下季度计划等？报告大约需要多长？

📎 可以上传文件让它分析

你
我上传了一个 Excel 表格，帮我计算各部门的总销售额，并生成一个新的汇总表
Claude
收到！我来读取你的表格……已完成分析，已生成汇总表，点击链接即可查看：销售汇总.xlsx

🔄 可以反复修改，直到满意

你
这份报告写得太正式了，语气改得轻松一点，像写给同事看的那种
Claude
明白！我来调整一下语气，让它更口语化、更亲切……
ℹ️
Claude 会主动问你问题如果你的需求不够清楚，Claude 会在开始工作前问你几个问题，帮它更准确地完成任务。这是正常现象，直接回答就好。

🌟 5   常见使用场景

以下是普通职场人最常用的几类任务，每种都附有可以直接复制使用的例子。

✨ 6   对话技巧与提示词写法

掌握以下几个技巧，可以让 Claude 更好地理解你的需求，输出更满意的结果。

技巧一：说明背景和受众

👍
好的写法 「帮我写一篇 面向非专业人士 的人工智能科普文章，约 800 字，语气轻松幽默」
👎
模糊的写法「写一篇 AI 的文章」——太笼统，Claude 不知道面向谁、写多长、什么风格。

技巧二：分步骤提问

如果任务比较复杂，不要一次性说完所有要求，可以分几轮对话逐步细化：

你（第1轮）
帮我做一份新员工入职培训方案的大纲
好的，这里是大纲……（列出6个章节）
你（第2轮）
第三章「公司文化」展开写详细内容，大约 300 字
好的，以下是第三章的详细内容……

技巧三：提供范例

如果你有参考样本，直接上传或粘贴给 Claude，让它照着格式来做，效果更准确。

💡
「我上传了一份去年的报告作为参考，请按照同样的格式和风格，帮我写今年的版本。」

技巧四：告诉它不要什么

🚫
「写邮件，不要太正式，不要超过 200 字，不要用英文单词」——用否定句约束范围，效果很好。

技巧五：直接说「不满意，重写」

如果 Claude 的输出不符合你的预期，直接反馈就好，比如：

  • 「太长了，压缩到一半」
  • 「换一个更有冲击力的标题」
  • 「重新排版，改成表格形式」
  • 「语气太客气了，直接说重点」

🔒 7   安全须知

Claude Cowork 非常注重你的数据安全，以下是几点重要提醒：

⚠️
不要输入敏感信息请避免将银行卡号、身份证号、密码、公司机密等高度敏感信息直接粘贴给 Claude。
✅
文件权限可控Claude 只能访问你主动选择授权的文件夹，不会偷偷读取你电脑上的其他内容。
ℹ️
高风险操作需要你确认涉及删除文件、发送邮件、提交表单等不可撤销的操作，Claude 都会在聊天框里请求你的明确确认，你说「是」或「确认」后才会执行。
🌐
网页内容需谨慎如果 Claude 访问了某个网页，而网页里有「奇怪的指令」，Claude 会暂停并告诉你，让你决定是否继续。

❓ 8   常见问题解答

Q：Claude Cowork 需要付费吗？

是的，Cowork 仅对付费用户开放，免费版无法使用。目前支持以下付费计划：

套餐月费（美元）说明
Pro$20 / 月可使用 Cowork，但用量较多时会较快触及限额
Max 5x$100 / 月使用量是 Pro 的 5 倍，适合高频用户
Max 20x$200 / 月使用量是 Pro 的 20 倍，适合重度日常用户
Team（高级席位）$125 / 月 / 席团队版需分配「高级席位」才能使用 Cowork

Cowork 的费用已包含在上述套餐中，无需额外付费。由于 Cowork 任务消耗的额度比普通对话多，Pro 用户在密集使用时可能会更快遇到限额提示。

Q：它支持中文吗？

完全支持！你可以全程用中文与 Claude 对话，它生成的文档、表格、PPT 也都可以是中文内容。

Q：生成的文件保存在哪里？

Claude 会将生成的文件保存到你选择授权的文件夹中，完成后会在聊天框里给你一个链接，点击即可直接打开。

Q：我的文件会上传到云端吗？

Claude 需要读取文件内容才能帮你处理。建议查阅 Anthropic 官方隐私政策，了解数据处理方式。对于高度机密的商业文件，建议咨询公司 IT 部门后再使用。

Q：Claude 会犯错吗？

会的！Claude 是 AI，它有时会产生不准确的内容，尤其是涉及具体数字、专业法律/医疗建议时。重要文件请务必自己核对一遍再使用。

Q：一次可以上传多个文件吗？

可以。你可以一次性上传多个文件，让 Claude 综合处理，比如「把这 3 份调研报告的核心结论汇总成一页摘要」。

Q：如果 Claude 不理解我的意思怎么办？

换一种方式描述你的需求，或者举一个具体的例子。比如「就像你在给小学生讲课一样，解释一下……」往往效果很好。
