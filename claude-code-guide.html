1什么是 Claude Code？

Claude Code 是 Anthropic 推出的一款 AI 编程助手，你可以通过"终端"（也叫"命令行"）来使用它。

但等等——我不会写代码啊！

没关系。Claude Code 的厉害之处在于，你只需要用自然语言告诉它你想做什么，它会帮你完成。比如：

  • 🗂️  "帮我整理一下这个文件夹里的图片"
  • 📊  "帮我分析这份 Excel 数据，找出规律"
  • 📝  "帮我写一段代码，自动发送邮件提醒"
  • 🔍  "帮我看看我的项目代码有没有问题"

Claude Code 就是你的 AI 程序员助手，而你是它的老板，用中文下达指令就行。

和 Claude 网页版有什么区别？

🌐 Claude 网页版

  • 在浏览器中聊天
  • 文件需手动上传 / 下载
  • 只给建议，不执行
  • 适合问答、写作

⚡ Claude Code（本指南）

  • 在终端中使用
  • 直接读写电脑文件
  • 可以直接执行代码
  • 自动化任务、处理文件

简单说：Claude Code 可以直接在你的电脑上"动手干活"，而网页版只能"动嘴说话"。

2准备 API Key

使用 Claude Code 需要一个 API Key（密钥）。共有三种方式获取，推荐优先考虑中转 API：

⭐ 推荐

🔀 方式一：第三方中转 API

由第三方服务商提供的 Claude API 代理，价格通常只有官方的 1/3 到 1/10，无需海外信用卡，按实际用量付费，国内网络即可访问。

✅ 无需海外信用卡服务稳定性依赖服务商

✅ 配置简单（两个变量）–

直接在 Anthropic 官网订阅，需要海外信用卡，价格相对较高，但最官方、最稳定。

8常见问题解答

输入 claude 提示"命令未找到"怎么办？

这是最常见的问题，通常只需重启终端即可解决：

macOS：

  1. 关闭当前终端窗口
  2. 重新打开终端
  3. 再次输入 claude

如果还是不行，在终端运行：

source ~/.zshrc

Windows：关闭 PowerShell，重新打开，再次输入 claude。

提示"需要 Pro 账号"怎么办？

你的账号可能是免费版，Claude Code 不支持免费账号。请访问 claude.ai/upgrade 升级到 Pro 或 Max 套餐。

能不能不用终端，直接用图形界面？

可以，但有限制。Claude Code 有官方桌面应用版：

  • macOS 版下载
  • Windows 版下载
⚠️ 重要限制：桌面应用仅支持 Claude Pro / Max 官方会员登录使用。
使用第三方中转 API、Anthropic Console API Key、Bedrock、Vertex AI 等方式的用户无法使用桌面端，请继续使用终端（命令行）版本。
Claude Code 2.0 启动时强制要求登录，用了 API Key 也没用怎么办？
✨
使用一键安装脚本可自动解决此问题！
如果你使用了第3节的一键安装脚本并选择了"中转 API"方式，脚本已经自动创建了 ~/.claude.json 配置文件，无需手动操作。
本节内容仅供手动安装的用户参考。

这是 Claude Code 2.0 的一个已知问题：即使正确设置了 ANTHROPIC_API_KEY 和 ANTHROPIC_BASE_URL，首次启动时仍会弹出登录界面，且会尝试连接 api.anthropic.com 进行验证，导致中转 API 用户无法通过。

手动解决方法：创建配置文件，跳过 onboarding 流程。

🍎 macOS / Linux：

在终端中运行以下命令（一次性执行即可）：

cat > ~/.claude.json << 'EOF'
{
  "hasCompletedOnboarding": true,
  "primaryApiKey": "你的API_Key"
}
EOF

将 你的API_Key 替换为你的实际 API Key，然后确保 ANTHROPIC_BASE_URL 也已设置（见第 2 节），再运行 claude 即可。

🪟 Windows（PowerShell）：

@"
{
  "hasCompletedOnboarding": true,
  "primaryApiKey": "你的API_Key"
}
"@ | Set-Content -Path "$env:USERPROFILE\.claude.json" -Encoding UTF8
💡 原理说明：Claude Code 2.0 在 ~/.claude.json 中用 hasCompletedOnboarding: true 判断是否跳过登录引导。手动写入此文件后，启动时便不再强制弹出登录界面，转而直接读取环境变量中的 API Key。

⚠️ 此为社区总结的变通方案，官方后续版本可能调整行为，若失效请检查是否有新版本的说明。
Claude Code 会乱删我的文件吗？安全吗？

Claude Code 默认在每次执行有影响的操作前会先询问你确认。比如要删除文件或修改代码时，它会先显示打算做什么，等你按 Y 确认后才执行。

建议在重要操作前做好备份，以防万一。

使用时会额外收费吗？
  • Claude Pro / Max 套餐：Claude Code 的使用包含在月费中，不额外收费。
  • Anthropic Console API：按 API 调用量计费，费用取决于使用量。
  • Amazon Bedrock / Vertex AI：按对应云平台的 AI 模型使用量计费。
可以用中文吗？

完全可以。Claude Code 支持中文对话，直接用中文描述需求即可，无需切换语言。

我没有编程经验，能用好 Claude Code 吗？

当然可以！Claude Code 本来就是为了让不懂编程的人也能完成自动化任务而设计的。

诀窍是：描述越具体，结果越准确。

与其说"整理文件"，不如说"把桌面上所有 2024 年之前创建的 PDF 文件移动到 Documents/旧文件 文件夹"。
